
## Project Title

Sample Blazor App to connect with Graph API for viewing calendar events and contacts



## API Reference

https://graph.microsoft.com

#### Get all contacts

```http
  GET /v1.0/me/contacts
```

#### Get all calendar events

```http
  GET /v1.0/me/events
```
